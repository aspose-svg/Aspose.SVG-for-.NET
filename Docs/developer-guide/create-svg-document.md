# Create SVG Documents in C#

**Aspose.SVG for .NET** lets you create SVG documents from scratch, strings, streams, files, and URLs. You can build SVG content with standard DOM APIs or use the fluent SVG Builder API for complex graphics.

Use this guide when you need to generate SVG dynamically in a .NET application, create SVG from C# code, or prepare an SVG document for editing, rendering, conversion, or saving.

## Create an Empty SVG Document

Create a blank `SVGDocument`, configure the root `<svg>` element, and add content with DOM methods.

```csharp
using Aspose.Svg;

using (var document = new SVGDocument())
{
    document.RootElement.SetAttribute("width", "300");
    document.RootElement.SetAttribute("height", "120");
    document.RootElement.SetAttribute("viewBox", "0 0 300 120");

    var circle = document.CreateElementNS("http://www.w3.org/2000/svg", "circle");
    circle.SetAttribute("cx", "60");
    circle.SetAttribute("cy", "60");
    circle.SetAttribute("r", "40");
    circle.SetAttribute("fill", "teal");
    document.RootElement.AppendChild(circle);

    document.Save("created.svg");
}
```

## Create SVG from a String

Use an SVG string when the markup is generated by another service, template, or user input.

```csharp
using Aspose.Svg;

string svgContent = "<svg xmlns='http://www.w3.org/2000/svg' width='200' height='100'><text x='20' y='60'>Hello SVG</text></svg>";

using (var document = new SVGDocument(svgContent, "."))
{
    document.Save("from-string.svg");
}
```

## Create SVG with the Builder API

The Builder API is useful for readable, fluent C# code when SVG content contains nested shapes, paths, styles, gradients, or reusable definitions.

```csharp
using Aspose.Svg;
using Aspose.Svg.Builder;
using System.Drawing;

using (var svgDocument = new SVGDocument())
{
    new SVGSVGElementBuilder()
        .Width(420)
        .Height(180)
        .ViewBox(0, 0, 420, 180)
        .AddRect(rect => rect
            .X(20).Y(20).Width(160).Height(100)
            .Fill(Color.Gold)
            .Stroke(Color.DarkOrange)
            .StrokeWidth(4))
        .AddCircle(circle => circle
            .Cx(280).Cy(70).R(50)
            .Fill(Color.Teal))
        .Build(svgDocument.RootElement);

    svgDocument.Save("builder.svg");
}
```

## Common Questions

### Can I create SVG without an input file?

Yes. Use `new SVGDocument()` and add elements to `document.RootElement`.

### Can I create SVG from generated markup?

Yes. Pass an SVG string and a base URI to the `SVGDocument` constructor.

### Should I use DOM APIs or SVG Builder API?

Use DOM APIs for direct document manipulation and targeted edits. Use Builder API when you want fluent, readable construction of nested SVG content.
