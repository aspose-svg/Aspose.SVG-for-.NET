# Supported Features - Aspose.SVG for .NET

**Aspose.SVG for .NET** is a C# API for creating, loading, editing, rendering, converting, vectorizing, and optimizing SVG documents in .NET applications.

Use this page to understand what Aspose.SVG for .NET can do and where to find examples for each workflow.

## What Can Aspose.SVG for .NET Do?

Aspose.SVG for .NET supports SVG document creation, DOM editing, CSS selection, XPath querying, rendering, conversion, image vectorization, text vectorization, SVG filters, gradients, transformations, color management, font handling, and fluent SVG creation with the Builder API.

## Common Questions

### Can Aspose.SVG for .NET convert SVG to PDF?

Yes. Use `Converter.ConvertSVG()` with `PdfSaveOptions` or rendering APIs to convert SVG documents to PDF.

### Can Aspose.SVG for .NET convert SVG to PNG or JPEG?

Yes. Aspose.SVG can render SVG to raster image formats such as PNG, JPEG, BMP, TIFF, and GIF.

### Can Aspose.SVG for .NET edit SVG files?

Yes. Load an `SVGDocument`, select elements with DOM APIs, CSS selectors, or XPath, update attributes or styles, and save the document.

### Can Aspose.SVG for .NET vectorize raster images?

Yes. The vectorization APIs can convert raster images to SVG paths and create stencil-style SVG output.

### Can Aspose.SVG for .NET create SVG from C# code?

Yes. You can create SVG with DOM APIs or with the fluent SVG Builder API.
